Arquivo bundle que contem todo as dependencias pre configuradas
Executar as seguintes ações:
1- alterar nome do projeto e versão  no arquivo package. json
2- alterar RewriteRule ./Reactpt3/public/index.html [L] em .htaccess
3- alterar BASE_URL: "/reactpt3/public" em src/Config.json
4- Executar projeto com o comando npm install
5- Iniciar o projeto com o comando gulp
